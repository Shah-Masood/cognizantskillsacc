Dataset Insights:
The dataset was taken from IMDB movie reviews.
It originally had binary labels but was expanded to include a neutral class.
The dataset was balanced to include equal samples per category.

Training Process:
Fine-tuned DistilBERT for sentiment classification.
Used 200 training samples and 50 test samples for quick testing.
Model trained for 3 epochs with a learning rate of 2e-5.

Evaluation Results:
Accuracy: ~85-90%
F1-score showed balanced classification across sentiment categories.

Application and Impact:
Can be used in customer support to classify feedback.
E-commerce platforms can analyze product reviews for sentiment trends.
Social media monitoring for sentiment analysis on brand perception.
